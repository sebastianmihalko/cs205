Web Page Programming and Design

This introduction to web development and web programming is intended for both CIS majors and non-majors. Topics include basic aspects of good web design and introductions to technologies that add functionality such as HTML 5, CSS 3, JavaScript, DOM (the document object model), XML, PHP, Ajax, and the MySQL database.
